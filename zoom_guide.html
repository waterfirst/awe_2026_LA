03-B
💻 Clone 상세 방법
① Git 설치 (처음 한 번만)
Windows: git-scm.com/download/win 접속 → 다운로드 → 설치
Mac: 터미널 열고 아래 입력 후 설치 팝업에서 "설치" 클릭:
git --version
설치 확인: 터미널에 git version 2.x.x 가 나오면 완료 ✅
② 클론 + 내 GitHub 레포로 연결
# 1. 템플릿 다운로드 git clone https://github.com/waterfirst/awe_2026_LA.git cd awe_2026_LA # 2. github.com/new 에서 새 레포 만든 후: git remote remove origin git remote add origin https://github.com/[내계정]/awe_2026_LA.git # 3. 내 레포에 업로드 git push -u origin main
③ 이후 파일 추가할 때마다
git add . git commit -m "사진 및 메모 추가" git push origin main
Push 할 때마다 GitHub Pages가 자동으로 업데이트됩니다 ✅

방법 2 — GitHub 웹 업로드

① index.html 업로드
1. github.com/[내계정]/awe_2026_LA 접속

1. 레포 상단 ⚙️ Settings 탭

3. Source: Deploy from a branch
4. Branch: main / Folder: / (root) → Save
5. 1~3분 대기 후 URL 생성 ✅
