01
🔑 Antigravity 가입
Google의 AI 코딩 도구입니다.
코딩을 몰라도 자연어로 지시하면 HTML을 만들어줍니다.
접속 URL
https://antigravity.google
가입 방법
1. 위 URL 접속
2. Sign in with Google 클릭
3. 회사 이메일 또는 개인 Gmail로 로그인
4. 완료 ✅
⚠️ 주의
무료 플랜은 토큰(사용량) 제한이 있습니다.
보고서 한 개 만들기에는 충분하지만, 큰 파일이 많으면 유료 플랜이 필요할 수 있습니다.
02
🐙 GitHub 계정 만들기
보고서를 올리고 URL을 만들 공간입니다.
평생 무료로 사용 가능합니다.
접속 URL
https://github.com → Sign up
가입 방법
1. github.com 접속 → Sign up
2. 이메일 / 비밀번호 / 사용자명 입력
3. 이메일 인증
4. 완료 ✅
💡 사용자명 추천
나중에 URL이 됩니다: [내계정].github.io/awe_2026_LA
알아보기 쉬운 영문 소문자 이름 추천 (예: kimyoungjae, leejinsoo)
03
🍴 템플릿 Fork
제가 만들어둔 AWE 보고서 템플릿을 여러분 계정으로 복사합니다.
템플릿 레포
github.com/waterfirst/awe_2026_LA
Fork 방법
1. 위 URL 접속
2. 우측 상단 Fork 버튼 클릭
3. Create fork 클릭
4. 내 계정에 복사 완료 ✅

→ 내 계정: github.com/[내계정]/awe_2026_LA
📋 포함 내용
• index.html — 4탭 보고서 템플릿 (일정/전시센싱/주목기업/최종소감)
• notes/ — 현장 메모 양식 파일
• img/ — 날짜별 사진 폴더
04
📁 현장 자료 정리
현지에서 이렇게 모으세요.
awe_2026_LA/ ├── img/ │ ├── day1/ ← 이동일 사진 │ ├── day2/ ← 전시 첫째날 사진 │ ├── day3/ ← 전시 둘째날 사진 │ └── day4/ ← 귀국일 └── notes/ ├── exhibition.txt ← 부스 관찰 메모 └── meetings.txt ← VIP 미팅 메모
현장 메모 팁
📱 카카오톡 나에게 보내기 → 텍스트 실시간 기록
📷 사진은 많이 찍을수록 좋음 (보고서가 풍성해짐)
📝 귀국 후 메모를 notes/exhibition.txt에 복붙
📝 exhibition.txt 작성 예시
[부스명: Meta]
- 제품: Quest 4 Pro 시연
- 특징: 시야각 120° · 무게 280g
- 고객 반응: 30분 대기줄
- 인상: 경쟁사 대비 가격 경쟁력 우수
05
🤖 Antigravity로 보고서 생성
자료가 모이면 Antigravity에서 index.html을 완성합니다.
사용 방법
1. antigravity.google 접속
2. 폴더(awe_2026_LA/) 전체 드래그&드롭 업로드
3. 아래 프롬프트 복붙 → 전송
4. 생성된 index.html 다운로드
-- 아래를 복붙하세요 -- notes/ 폴더 메모와 img/ 폴더 사진을 바탕으로 index.html 출장보고서를 완성해줘. 규칙: 1. 탭 구성 유지 (일정/전시센싱/주목기업/최종소감) 2. [ ] 플레이스홀더를 실제 내용으로 교체 3. img/ 사진을 날짜별로 해당 섹션에 배치 4. exhibition.txt 내용을 전시센싱 탭에 반영 5. meetings.txt 내용을 주목기업 탭에 반영 6. 단일 HTML 파일로 출력 7. 다크 테마와 탭 구조 유지
06
🚀 GitHub Pages 배포
생성된 index.html을 GitHub에 올리면 URL이 생깁니다.
업로드 방법
1. github.com/[내계정]/awe_2026_LA 접속
2. Add file → Upload files
3. Antigravity에서 받은 index.html 드래그
4. Commit changes 클릭
Pages 활성화 (최초 1회)
1. 레포 → Settings 탭
2. 왼쪽 메뉴 → Pages
3. Source: main branch → / (root) → Save
4. 1~2분 후 URL 생성 ✅
내 보고서 URL
https://[내계정].github.io/awe_2026_LA
BONUS
📷 사진 업로드 (선택)
사진을 보고서에 실제로 표시하려면 img/ 폴더에도 올려야 합니다.
사진 업로드 방법
1. github.com/[내계정]/awe_2026_LA
2. img/day2/ 폴더 클릭
3. Add file → Upload files
4. 사진 여러 장 선택 → Commit
5. Antigravity에 다시 업로드 → 보고서 재생성
💡 팁: GitHub에서 직접 편집도 가능
index.html → 연필(✏️) 아이콘 클릭 → 내용 수정 → Commit
간단한 텍스트 수정은 Antigravity 없이도 가능합니다
