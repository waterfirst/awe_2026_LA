Section 02-B — Antigravity

Antigravity — 바이브코딩으로
보고서·앱 만들기

코딩 없이 자연어 지시만으로 인터랙티브 HTML 보고서와 웹앱을 만듭니다.
이번 AWE 출장 보고서도 Antigravity로 만들었습니다.

폴더 통째로 올리기

사진 폴더 + 메모 파일을 드래그하면 AI가 전체 맥락을 파악하고 완성된 HTML을 자동 생성합니다.

출장 보고서, 전시 리포트, 프로젝트 요약 자료에 적합

자연어 지시로 수정

"3번 탭에 사진 추가해줘", "다크 테마로 바꿔줘", "차트 넣어줘" — 말하면 즉시 반영됩니다.

디자인·코딩 지식 없어도 완성도 높은 결과물

Publish로 즉시 공유

완성된 결과물을 Publish 버튼으로 배포하면 URL이 즉시 생성됩니다. Firebase(.web.app) 또는 Cloud Run(.run.app) 도메인으로 공유 가능.

설치 없이 링크 하나로 팀 전체 열람

Base64 사진 내장

"모든 사진을 Base64로 내장해줘"를 프롬프트에 추가하면 HTML 파일 하나에 사진이 모두 포함됩니다.

이메일 첨부, USB 전달, GitHub 업로드 간소화
실습 프롬프트 예시 (복붙 사용)
폴더 안의 사진과 메모를 바탕으로 출장 보고서 HTML을 완성해줘.
규칙: 탭 구성 유지, 모든 [ ] 플레이스홀더 교체, 사진 날짜별 배치, 모든 사진 Base64 내장, 단일 HTML 파일 출력, 다크 테마 유지.
