# AWE 2026 LA 출장 보고서

> AR/XR 기술 동향 출장 보고서 — AI(Antigravity) 기반 자동 생성 템플릿

## 🚀 빠른 시작

1. **이 레포 Fork** → 우측 상단 Fork 버튼
2. **현장에서 자료 수집** → `notes/` 메모 + `img/` 사진
3. **Antigravity로 생성** → [antigravity.google](https://antigravity.google)
4. **index.html 업로드** → GitHub에 커밋
5. **GitHub Pages 활성화** → Settings → Pages → main / (root)

내 보고서 URL: `https://[내계정].github.io/awe_2026_LA`

---

## 📁 폴더 구조

```
awe_2026_LA/
├── index.html          ← 보고서 본체 (4탭: 일정/전시센싱/주목기업/최종소감)
├── zoom_guide.html     ← Zoom 교육 슬라이드 (11장)
├── img/
│   ├── day1/           ← 이동일 사진
│   ├── day2/           ← 전시 첫째날
│   ├── day3/           ← 전시 둘째날
│   └── day4/           ← 귀국일
└── notes/
    ├── exhibition.txt  ← 부스 관찰 메모 양식
    └── meetings.txt    ← VIP 미팅 메모 양식
```

---

## 🤖 Antigravity 프롬프트 (복붙용)

```
notes/ 폴더 메모와 img/ 폴더 사진을 바탕으로
index.html 출장보고서를 완성해줘.

규칙:
1. 탭 구성 유지 (일정/전시센싱/주목기업/최종소감)
2. [ ] 플레이스홀더를 실제 내용으로 교체
3. img/ 사진을 날짜별로 해당 섹션에 배치
4. exhibition.txt 내용을 전시센싱 탭에 반영
5. meetings.txt 내용을 주목기업 탭에 반영
6. 단일 HTML 파일로 출력
7. 다크 테마와 탭 구조 유지
```
